#Funcionalidad del Programa de Gestion de Paises
-Muestra países con sus respectivos datos
-Permite agregar países
-Permite buscar países
-Filtra los países por continente
-Permite ordenar según lo requerido (Nombre, Población, Superficie)
-Muestra estadísticas

#Estructuras Utilizadas
-Archivo CSV
-Listas
-Diccionarios
-Funciones
-Ciclos for
-Ciclo While
-Uso de if, elif, else
-Uso de try y except para manejo de errores
